# RDBMS

A relational database project built on Oracle (PL/SQL) designed to model, manage, and track the complex ecosystem of a professional football club; from squads, contracts, and sponsorships to match fixtures, stadiums, and real-time statistics such as goals, assists, and cards. The system exposes three role-based interfaces implemented as PL/SQL packages: a Club Manager for overseeing club assets, contracts, and personnel; a Match Analyst for querying match results, events, and performance metrics; and a Player Agent for managing player profiles, sponsorships, and season statistics. Complementing the core schema, the project includes indexes for query optimization, views for common reporting queries, triggers for data integrity enforcement, stored procedures that dynamically calculate league standings and generate match-day reports, and a pre-populated insert script for testing.
