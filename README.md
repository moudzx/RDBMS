# RDBMS

A relational database project designed to model, manage, and track the complex ecosystem of a professional football league. Utilizing a structured schema of interconnected tables, the system efficiently handles data normalization for fundamental league entities including teams, players, match fixtures, referees, stadiums, and real-time statistics (such as goals, assists, cards, and substitutions). Built to demonstrate robust data modeling and relational integrity, the project includes optimized SQL queries, views, and stored procedures that dynamically calculate league standings, track player performance metrics, and generate detailed match-day reports.
